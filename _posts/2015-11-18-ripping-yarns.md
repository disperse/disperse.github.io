Ripping Yarns is a mod I am developing for Mount & Blade Warband. The idea is, with procedural generation and 
permadeath, to create a rogue-like game mode that can be completed in a single evening.

## Immediate goals ##
- Procedurally generate new map.txt and parties.txt on each run of the mod
- Greatly increase experience point rate

## Short-term goals ##
- Add permadeath
- Remove unit respawn to prevent farming

## Long-term goals ##
- Daily runs
- Voluntary Challenges
  - Solo run
